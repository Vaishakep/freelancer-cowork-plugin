# /freelancer:project-retro

Run a full retrospective on a completed project.

## Instructions

1. Ask the user about the project: client, deliverables, timeline, what they charged, hours spent.
2. Ask what went well and what was challenging.
3. Load the project-retro skill and generate all three outputs:
   - Lessons document
   - Case study draft
   - Pricing intelligence note
4. Offer to format the case study for LinkedIn or their portfolio site.
5. Offer to draft a testimonial request email to the client.
