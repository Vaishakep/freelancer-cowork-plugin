# /create-sow

Generate a professional Scope of Work document that protects against scope creep.

## Instructions

1. Ask the user for the project name, client name, and a brief description of the work.
2. Ask about deliverables, timeline, pricing, and revision rounds.
3. Follow the scope-of-work skill structure to generate the full SOW:
   - Project overview
   - Numbered deliverables with measurable outputs
   - Out of scope section
   - Timeline and milestones table
   - Revisions and feedback policy
   - Investment and payment terms
   - Communication expectations
   - Ownership and rights
   - Acceptance/signature block
4. Generate the complete SOW in clean, professional format.
5. Offer to export as a Word document or PDF for the user to send.
