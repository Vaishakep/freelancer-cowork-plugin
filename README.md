# Freelancer Plugin for Claude Cowork

Turn Claude into your freelance business partner. Built for solopreneurs, independent consultants, and anyone running a one-person business.

## What It Does

This plugin gives Claude deep expertise in the business side of freelancing — the stuff that isn't your actual craft but takes up half your time.

**6 Skills** that activate automatically when relevant:
- **Proposal Writer** — Generate client-winning proposals from job postings or intake notes
- **Client Comms** — Draft any client message with the right tone (updates, pushback, bad news, testimonial requests)
- **Scope of Work** — Create professional SOW documents that prevent scope creep
- **Invoice Follow-Up** — Escalating payment reminders from friendly nudge to final notice
- **Project Retrospective** — Turn completed projects into lessons, case studies, and pricing data
- **Rate Negotiation** — Handle pricing objections, set rates, and raise prices confidently

**5 Slash Commands** for when you need something specific:
- `/freelancer:write-proposal` — Generate a proposal
- `/freelancer:create-sow` — Build a Scope of Work
- `/freelancer:chase-payment` — Draft a payment follow-up
- `/freelancer:project-retro` — Run a project retrospective
- `/freelancer:pricing-help` — Get rate and pricing guidance

## Install

### From GitHub
```bash
claude plugin marketplace add Vaishakep/freelancer-cowork-plugin
claude plugin install freelancer@freelancer-cowork-plugin
```

### Manual Install
1. Download or clone this repository
2. Open Claude Desktop → Cowork tab
3. Click Customize → Browse plugins → Upload
4. Select the plugin folder

## Customize It

After installing, click "Customize" on the plugin in Cowork. Tell Claude about:
- Your niche/industry
- Your typical project types and price ranges
- Your preferred proposal style
- Your payment terms

Claude will tailor the skills to match how YOU work.

## Who This Is For

- Freelance developers, designers, writers, consultants
- Independent contractors and solopreneurs
- Small agency owners (1-5 people)
- Anyone who trades time/expertise for money and needs help with the business side
